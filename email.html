收藏与订阅功能演示

轻量级功能组件，只需引入脚本即可在您的网站中集成邮件订阅和页面收藏功能

邮件订阅功能

简单集成即可让用户订阅您的内容更新，支持邮箱验证和订阅管理

调用方式

<!-- HTML结构 -->
<div class="my-custom-subscription">
  <input type="email" id="subscribeEmail">
  <button id="subscribeBtn">订阅</button>
  <div id="subscriptionMessage"></div>
</div>

<!-- 仅需引入此脚本 -->
<script type="module" src="/main.js"></script>

自定义通知方式 

// 在引入main.js前定义
window.notificationConfig = {
  // 通知显示位置: top-left, top-right, bottom-left, bottom-right
  position: 'bottom-right',
  // 通知显示时长(毫秒)
  duration: 3000,
  // 自定义通知模板
  template: (message, type) => `
    <div class="custom-notification ${type}">
      ${message}
    </div>
  `
};
