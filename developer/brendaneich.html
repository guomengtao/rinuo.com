创造了世界上最流行的编程语言之一，推动了现代 web 开发的革命，Mozilla 与 Firefox 的关键领导者。

职业履历

Brave Software 创始人兼CEO

创立了专注于隐私保护的浏览器公司，开发了Brave浏览器，引入了基于区块链的注意力经济系统，彻底改变了用户与广告商的互动方式。

负责公司战略方向，技术创新和产品开发
2015年 - 至今

Mozilla 公司 CEO

担任Mozilla公司首席执行官，领导公司继续推动开放网络和互联网隐私保护的使命。尽管任期短暂，但对公司战略产生了重要影响。

2014年

Mozilla 公司 CTO

担任Mozilla首席技术官，领导Firefox浏览器的技术开发和创新，推动开放网络标准，确保互联网的开放性和可访问性。

2005年 - 2014年

Mozilla 项目联合创始人

与其他人共同创立了Mozilla项目，这是一个致力于保持互联网健康和开放的非营利组织，负责维护和发展Firefox浏览器及相关技术。

1998年

创造 JavaScript

在网景公司工作期间，仅用10天时间设计并实现了JavaScript编程语言，最初名为Mocha，后更名为LiveScript，最终定名为JavaScript。这一发明彻底改变了网页的交互性。

1995年

网景通信公司 工程师

加入网景公司，担任工程师，负责开发网景 Navigator 浏览器的相关技术，为后来JavaScript的诞生奠定了基础。

1994年 - 1998年

主要项目

JavaScript

世界上最流行的编程语言之一，用于为网页添加交互性。最初为网景浏览器设计，现已成为前端开发的核心语言，并通过Node.js扩展到后端开发。

编程语言 Web开发 前端 后端

Mozilla

致力于保持互联网开放和可访问的非营利组织，开发了Firefox浏览器，推动开放标准，并倡导网络隐私和数字权利。

非营利组织 开源 浏览器

Firefox

由Mozilla开发的开源网页浏览器，以其速度、安全性和对用户隐私的保护而闻名，是市场上主要的浏览器之一。

浏览器 开源 隐私保护

Brave浏览器

专注于隐私保护的网页浏览器，内置广告拦截功能，引入了基于区块链的BAT（Basic Attention Token）系统，重新定义了内容创作者的收入模式。

浏览器 隐私保护 区块链

成功秘诀

我认为成功来自于解决实际问题的能力，保持对技术的热情，以及愿意不断学习和适应变化的心态。JavaScript的诞生正是为了解决当时网页交互性不足的实际问题。
