Mocha

JavaScript 测试框架 · Node.js & 浏览器支持

分享到 Twitter

名字含义与起源

Mocha（摩卡）是一款灵活的 JavaScript 测试框架，运行在 Node.js 和浏览器环境中。名字取自咖啡“摩卡”，寓意开发者需要一杯咖啡来写测试 ☕。

由 TJ Holowaychuk 于 2011 年创建，是最早流行的 JS 单元测试框架之一。

优点与缺点

  • 优点：灵活、插件丰富、支持多种断言库、社区成熟。
  • 优点：支持异步测试，兼容 Node.js 和浏览器。
  • 缺点：配置略复杂，新手需要学习成本。
  • 缺点：相比 Jest，缺少内置断言和快照测试。

免费获取与使用方法

Mocha 是完全开源免费的，可以通过 npm 安装：

npm install --save-dev mocha

官方网站：https://mochajs.org

  • 👎 “对新手不友好，需要额外配置断言库。”
