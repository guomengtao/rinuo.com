scrapfly = ScrapflyClient(key="YOUR_API_KEY")
result = scrapfly.scrape(ScrapeConfig(
    url="https://www.tiktok.com/@kevinhart4real",
    render_js=True,
    proxy_pool="public_residential_pool"
))
print(result.content)

示例代码
# 安装
pip install playwright
playwright install

# 示例代码
from playwright.sync import sync_playwright

with sync_playwright() as p:
    browser = p.chromium.launch(headless=False)
    page = browser.new_page()
    page.goto("https://www.tiktok.com/@kevinhart4real")
    page.wait_for_selector("div.tiktok-1soki6x-DivItemContainer")
    videos = page.query_selector_all("div.tiktok-1soki6x-DivItemContainer")
    for video in videos[:5]:
        print(video.text_content())
    browser.close()

非技术用户

推荐使用无代码工具如Apify TikTok Scraper或Axiom.ai，上手快，无需编程知识。

开发者

根据需求选择API服务或开源脚本，API适合快速集成，开源脚本适合定制化需求。

企业用户

建议选择企业级方案如Bright Data或Oxylabs，保证稳定性和合规性，适合大规模数据采集。
