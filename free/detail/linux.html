Cool Linux Commands

ls = list files

This command shows you all the files in your folder, like looking inside your toy box!

cd games = change directory

This takes you into the "games" folder, like opening a door to a new room!

echo "Hi" = say something

This makes the computer repeat what you write, like playing echo with a friend!
