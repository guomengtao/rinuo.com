What is HTML?

HTML is like the building blocks for websites! It's the code that tells computers what to show on a webpage.

Think of HTML as instructions for drawing pictures, writing stories, and adding buttons to a page.

With HTML, you can create your own webpages with text, images, and fun stuff!

HTML Looks Like This:

<h1>My First Page</h1>
<p>Hello, world!</p>
<img src="cat.jpg">
<button>Click Me!</button>
