Swagger Code Example

// This is like a menu that shows what the API can do
swagger: '2.0'
info:
  title: 'Ice Cream API'
  description: 'An API for delicious ice cream flavors'
  version: '1.0.0'
paths:
  /flavors:
    get:
      summary: 'Get all ice cream flavors'
      responses:
        200:
          description: 'A list of yummy flavors'
          schema:
            type: 'array'
            items:
              type: 'string'
              examples:
                - 'Chocolate'
                - 'Vanilla'
                - 'Strawberry'

This Swagger code creates a "menu" for an Ice Cream API that shows you can get a list of flavors like Chocolate, Vanilla, and Strawberry!
