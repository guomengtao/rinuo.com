What is Prettier?

Prettier is an opinionated code formatter that supports many languages and integrates with most editors. It removes all original styling and ensures that all outputted code conforms to a consistent style.

Before Prettier

function greeting(name) {
  return "Hello, " + name + "!";
}

const user = {name: "John", age: 30}

if (user.age > 18) {
console.log("Adult")
}

After Prettier

function greeting(name) {
  return `Hello, ${name}!`;
}

const user = { name: "John", age: 30 };

if (user.age > 18) {
  console.log("Adult");
}

The Creator

Prettier was created by James Long, a talented developer with a vision for consistent code formatting.

James Long

Software Engineer, Creator of Prettier

James Long is a software engineer with a passion for developer tools and experience working at companies like Facebook and Shopify. He created Prettier out of frustration with code reviews focusing on formatting rather than logic and functionality.

His vision was to create a tool that would eliminate debates about code style by enforcing a consistent format automatically, allowing developers to focus on what matters most - solving problems.

Project Origin

The story behind Prettier and how it revolutionized code formatting.

The Frustration

In 2016, James Long was working on projects where code reviews often got stuck on formatting debates rather than the actual logic. He noticed that developers were wasting valuable time arguing about braces, indentation, and line breaks.

The Idea

James had a vision for a tool that would take care of code formatting automatically, leaving no room for debates. Unlike linters that only warn about style issues, this tool would actually rewrite code to match a consistent style.

The First Release

Prettier was first released in October 2016. It started with JavaScript support but quickly expanded to other languages. The reception was overwhelming, with developers embracing the "no configuration" approach to code formatting.

The Evolution

Today, Prettier is used by millions of developers and thousands of companies worldwide. It has a thriving community of contributors and supports more than 20 programming languages and frameworks.

Resources & Installation

Get started with Prettier in your project in just a few simple steps.

Installation

Using npm

npm install --save-dev --save-exact prettier

Using yarn

yarn add --dev --exact prettier

Basic Usage

Format a file

npx prettier --write src/index.js

Format all files in a directory

npx prettier --write "src/**/*.{js,jsx}"
