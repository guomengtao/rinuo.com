Notion API Example 
import { Client } from '@notionhq/client';

// Initialize client
const notion = new Client({
  auth: process.env.NOTION_API_KEY,
});

// Create a new page
async function createPage() {
  const response = await notion.pages.create({
    parent: {
      database_id: 'DATABASE_ID'
    },
    properties: {
      Name: {
        title: [
          {
            text: {
              content: 'New Task'
            }
          }
        ]
      },
      Status: {
        select: {
          name: 'To Do'
        }
      }
    }
  });
  
  return response;
}

What is Notion API?

The Notion API is a powerful interface that allows developers to programmatically interact with Notion workspaces, databases, pages, and blocks. It enables you to create, read, update, and delete content within Notion, as well as automate workflows and build custom integrations.

Launched in 2021, the Notion API provides a structured way to extend Notion's functionality beyond its native capabilities. It uses REST principles and JSON for data exchange, making it accessible to developers across various programming languages and platforms.

With the Notion API, you can build applications that seamlessly integrate with Notion, from simple automation scripts to complex custom tools that enhance productivity and collaboration.

Basic API Example

Here's a simple example using Node.js to fetch a database:

// Initialize Notion client
import { Client } from '@notionhq/client';

const notion = new Client({
  auth: process.env.NOTION_API_KEY,
});

// Fetch database entries
async function fetchDatabase(databaseId) {
  const response = await notion.databases.query({
    database_id: databaseId,
  });
  
  return response.results;
}

// Example usage
fetchDatabase('your-database-id')
  .then(results => console.log(results))
  .catch(console.error);
