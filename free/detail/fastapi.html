Let's Try Some Code!

1 The Simplest FastAPI App

This is like building a tiny house for your code. It's the smallest FastAPI program you can make!

main.py
from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def read_root():
    return {"Hello": "World"}

Success! Your app says:

{ "Hello": "World" }

2 Making It Do Something Fun

This code creates a web page that can add numbers together. Cool, right?

main.py
from fastapi import FastAPI

app = FastAPI()

@app.get("/add/{num1}/{num2}")
def add_numbers(num1: int, num2: int):
    result = num1 + num2
    return {
        "message": "Adding numbers is fun!",
        "result": result
    }

Try adding two numbers:

+

Wow! Here's what happened:

{ "message": "Adding numbers is fun!", "result": 8 }
