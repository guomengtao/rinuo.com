How Do Programmers Use Me?

1. Set Up Django

Programmers first install Django on their computer, like opening a new building set box and getting ready to build.

# Programmers type this to install Django
pip install django
# They create a new project
django-admin startproject mycoolwebsite

2. Create a Project

They start a new project, which is like laying the foundation for their website building.

3. Build Website Parts

They add different parts to their website, like pages, forms, and pictures - just like adding rooms to a house!

# They create different parts
python manage.py startapp blog
python manage.py startapp photos
# They start their website
python manage.py runserver

# Then visit: http://localhost:8000

4. Run the Website

They start their website and can see it on their computer, making changes until it's perfect!
