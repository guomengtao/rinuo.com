名字的含义

Django

Django 这个名字来源于著名的爵士吉他手 Django Reinhardt：

  • 他是20世纪最伟大的吉他手之一，尽管左手受伤，仍然创造了独特的演奏风格
  • 名字体现了框架的优雅和强大特性
  • 与Python的命名传统一致（Python来自Monty Python剧团）

就像Django Reinhardt用两根手指创造音乐奇迹一样，Django框架让你用更少的代码创造强大的Web应用。

起源与发展

2003

诞生

Django 诞生于2003年，由 Adrian Holovaty 和 Simon Willison 在 Lawrence Journal-World 报社开发。他们需要快速开发新闻网站，因此创建了这个框架。

2005

开源发布

2005年7月，Django 以 BSD 许可证开源发布。这个名字正式确定，并开始吸引全球开发者的关注。

2008

Django软件基金会

成立了 Django Software Foundation，确保框架的长期发展和维护，标志着Django进入成熟发展阶段。

现在

行业标准

如今，Django 已成为 Python Web 开发的事实标准，被 Instagram、Mozilla、Pinterest 等大公司采用，拥有庞大的生态系统。

核心特性

Django 提供了一整套工具和功能，让Web开发变得简单高效

ORM对象关系映射

强大的ORM系统，支持多种数据库，让数据库操作变得简单直观。

内置安全机制

自动处理SQL注入、XSS、CSRF等安全问题，保护应用免受常见攻击。

管理后台

自动生成功能完善的管理界面，让内容管理变得轻而易举。

DRY原则

Don't Repeat Yourself - 鼓励代码复用，减少重复开发工作。

丰富插件生态

大量的第三方包和插件，扩展框架功能，满足各种需求。

活跃社区

全球开发者社区支持，丰富的文档和教程资源。

MTV架构模式

Django采用MTV（Model-Template-View）架构模式，让代码组织更清晰

Template (模板)

负责呈现用户界面，处理展示逻辑

View (视图)

处理业务逻辑，协调模型和模板

快速开始

安装 Django

使用 pip 安装 Django：

pip install Django

创建项目

创建一个新的 Django 项目：

django-admin startproject mysite
cd mysite
python manage.py runserver

提示：访问 http://127.0.0.1:8000/ 查看你的第一个 Django 应用！

代码示例

看看 Django 如何让 Web 开发变得简单优雅

Model 定义 

from django.db import models

class Article(models.Model):
    title = models.CharField(max_length=200)
    content = models.TextField()
    author = models.CharField(max_length=100)
    created_at = models.DateTimeField(auto_now_add=True)
    published = models.BooleanField(default=False)

    def __str__(self):
        return self.title

View 处理 

from django.shortcuts import render
from .models import Article

def article_list(request):
    articles = Article.objects.filter(
        published=True
    ).order_by('-created_at')
    
    return render(request, 'articles/list.html', {
        'articles': articles
    })

URL 路由配置 

from django.urls import path
from . import views

urlpatterns = [
    path('articles/', views.article_list, name='article_list'),
    path('articles/<int:pk>/', views.article_detail, name='article_detail'),
]
