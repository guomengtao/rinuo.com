Vue.js 使用方法

快速入门 Vue.js，从安装到创建第一个应用

步骤 1: 安装 Vue.js

有多种方式可以安装 Vue.js，选择最适合你项目的方式：

通过 CDN 引入

<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

适合快速原型开发，无需复杂配置

使用 npm 或 yarn

npm install vue@latest
# 或者
yarn add vue@latest

适合大型项目，便于管理依赖

使用 Vue CLI

npm install -g @vue/cli
# 或者
yarn global add @vue/cli

vue create my-project
cd my-project

官方推荐的脚手架工具，提供完整的项目配置

步骤 2: 创建 Vue 实例

安装完成后，创建一个基本的 Vue 实例：

HTML 文件

<div id="app">
  <h1>{{ message }}</h1>
</div>

JavaScript 文件

// 从 Vue 中导入 createApp 函数
import { createApp } from 'vue'

// 创建一个应用实例
const app = createApp({
  // 数据 - 函数返回一个对象
  data() {
    return {
      message: 'Hello, Vue!'
    }
  }
})

// 将应用挂载到 DOM 元素上
app.mount('#app')

这个简单的实例会在页面上显示 "Hello, Vue!"，使用了 Vue 的数据绑定功能。

步骤 3: 模板语法

Vue 使用基于 HTML 的模板语法，允许你声明式地将 DOM 绑定到 Vue 实例的数据：

文本插值

<!-- 文本插值 -->
<p>{{ message }}</p>

<!-- 一次性插值，数据变化不会更新 -->
<p v-once>{{ message }}</p>

HTML 绑定

<!-- 输出 HTML，会被解析为 HTML 而非文本 -->
<div v-html="rawHtml"></div>

指令

<!-- 条件渲染 -->
<p v-if="seen">现在你看到我了</p>

<!-- 列表渲染 -->
<ul>
  <li v-for="item in items" :key="item.id">
    {{ item.text }}
  </li>
</ul>

<!-- 事件监听 -->
<button v-on:click="doSomething">点击我</button>

步骤 4: 组件基础

组件是 Vue 应用的基本构建块，以下是创建和使用组件的示例：

创建组件

// 定义一个名为 button-counter 的新组件
const ButtonCounter = {
  data() {
    return {
      count: 0
    }
  },
  template: `<button v-on:click="count++">点击了 {{ count }} 次</button>`
}

注册和使用组件

// 全局注册组件
app.component('button-counter', ButtonCounter)

// 在模板中使用
<div id="app">
  <button-counter></button-counter>
  <button-counter></button-counter>
</div>

每个组件都是独立的，点击不同的按钮会各自增加计数，不会相互影响。

步骤 5: 运行应用

使用 Vue CLI 创建的项目，可以通过以下命令运行开发服务器：

启动开发服务器

npm run serve
# 或者
yarn serve

开发服务器默认运行在 http://localhost:8080

构建生产版本

npm run build
# 或者
yarn build

构建完成的文件会生成在 dist 目录下，可直接部署到服务器
