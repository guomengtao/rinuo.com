Let's Create a Video Game Collection!

Imagine we want to keep track of all our favorite video games. We can use PostgreSQL to organize them!

Step 1: Create a game collection

CREATE TABLE my_games ( id INT, name VARCHAR(100), genre VARCHAR(50), rating INT );

Step 2: Add some games

INSERT INTO my_games VALUES (1, 'Super Adventure', 'Adventure', 5), (2, 'Racing Stars', 'Racing', 4), (3, 'Puzzle Masters', 'Puzzle', 5);

Step 3: Find our favorite games (rating 5!)

SELECT name FROM my_games WHERE rating = 5;
What we get:
  • Super Adventure
  • Puzzle Masters
