What is 7-Zip?

7-Zip is like a magical backpack for your computer files! It can squish big files to make them smaller.

Imagine taking a big pillow and squishing it into a tiny bag - that's what 7-Zip does for computer files!

  • Makes big files take less space
  • Packs many files into one zip folder
  • Opens zipped files like opening a present

How Computers Do It

This is what 7-Zip tells the computer to do (in kid-friendly code!): 

// 7-Zip magic code
function zipFiles(files) {
  // Make files smaller by squishing them
  let squishedFiles = files.map(file => {
    return squishFile(file); // Like squeezing a sponge!
  });
  
  // Put all squished files into one zip folder
  let zipFolder = putIntoZipFolder(squishedFiles);
  
  // Ta-da! Here's your zip file
  return zipFolder;
}

// Let's try it with our files
let myFiles = [drawing.png, story.txt, song.mp3];
let myZip = zipFiles(myFiles);

// Wow! 8MB became only 3MB!
console.log("Your zip file is ready: " + myZip);
