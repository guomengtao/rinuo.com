What is Hoppscotch?

Hoppscotch is like a magical messenger that helps you talk to computers all around the world! When you visit websites or play online games, there are computers called "servers" working behind the scenes.

With Hoppscotch, you can send messages to these servers and see how they respond. It's a fun way to learn how the internet works while creating your own little experiments!

In simple words:

Hoppscotch helps you send questions to websites and see their answers. It's like having a secret conversation with the internet!

It Talks to Servers

Send special messages to computers that run websites and apps, and get replies back instantly!

It's Super Easy

You don't need to be an expert! Hoppscotch makes it simple to create your own internet experiments.

How Do You Use It?

Using Hoppscotch is as easy as 1-2-3! Follow these simple steps to start your internet adventure.

1

Pick Your Message Type

Choose what kind of message you want to send. The most common one is "GET" to ask for information.

GET
2

Where to Send It

Type in the address of the server you want to talk to, like you would type a website.

https://example.com/api/greet
3

Send & See Response

Click the send button and watch as the server sends back its reply!
