Netlify 的优点与缺点

了解Netlify平台的主要优势和潜在局限性，帮助您做出更明智的选择

主要优点

使用简单直观

界面友好，部署流程简单，即使是新手也能快速上手，无需复杂配置即可完成网站部署。

出色的性能表现

全球分布式CDN确保网站在世界各地都能快速加载，自动优化资产提升访问速度。

强大的免费计划

免费计划包含了丰富的功能，足以满足个人项目和小型网站的需求，无需信用卡即可开始使用。

无缝的Git集成

与主流Git平台深度集成，实现自动化部署流程，推送代码即可完成部署，极大提高开发效率。

优秀的开发者体验

提供详细的部署日志、即时反馈和便捷的回滚功能，让开发者能够快速定位和解决问题。

丰富的生态系统

与众多静态站点生成器、CMS系统和开发工具无缝集成，提供一致的开发体验。

强大的社区支持

拥有活跃的开发者社区和完善的文档，问题能快速得到解答，学习资源丰富。

主要缺点

免费计划有流量限制

免费账户每月有100GB的带宽限制，超过后会被暂停服务或强制升级付费计划。

Serverless函数限制

免费计划中的Serverless函数有调用次数和执行时间限制，不适合高负载的后端功能。

不适合动态网站

主要针对静态网站优化，虽然支持Serverless函数，但对于复杂的动态网站功能支持有限。

高级功能需付费

如高级分析、表单处理、身份验证等功能需要升级到付费计划才能使用。

自定义配置有限

相比自建服务器，Netlify的配置选项有限，某些高级服务器配置无法实现。

依赖第三方服务

作为托管平台，用户对基础设施的控制有限，平台故障可能影响网站可用性。

大型项目成本高

对于流量大或功能复杂的项目，升级到合适的付费计划可能成本较高。

Netlify 入门使用方法

按照以下步骤开始使用Netlify部署和托管您的网站

2
创建站点

域名设置

高级配置

步骤 1: 注册Netlify账号

首先，您需要注册一个Netlify账号，您可以使用以下方式之一注册：

  • 使用GitHub账号 - 推荐方式，便于后续与代码仓库集成
  • 使用GitLab或Bitbucket账号 - 同样支持与对应代码仓库集成
  • 使用电子邮件 - 创建独立的Netlify账号

访问 Netlify注册页面，选择您偏好的注册方式并完成注册流程。

步骤 2: 创建新站点

注册并登录后，您可以通过以下方式之一创建新站点：

从Git仓库导入

这是推荐的方式，选择已连接的Git账号中的仓库，Netlify将自动部署您的网站。

  1. 点击仪表盘中的"New site from Git"按钮
  2. 选择您的代码托管平台（GitHub、GitLab或Bitbucket）
  3. 授权Netlify访问您的仓库
  4. 选择您要部署的仓库

手动上传文件

适合简单的静态网站，直接上传HTML、CSS和JavaScript文件。

  1. 点击仪表盘中的"Deploy manually"按钮
  2. 拖拽您的网站文件到上传区域，或点击选择文件
  3. 等待上传完成，Netlify将自动部署您的网站

步骤 3: 配置部署设置

从Git仓库导入时，需要配置构建和部署选项：

分支设置

选择要部署的分支（通常是main或master），每次推送到该分支都会触发自动部署。

构建命令

输入构建您项目的命令，例如：

# 对于React应用

npm run build

# 对于Vue应用

npm run generate

# 对于静态HTML或不需要构建的项目，留空

发布目录

指定构建后生成的静态文件所在目录，例如：

# 对于React应用

build

# 对于Vue应用

dist

# 对于直接在根目录的静态文件

.

配置完成后，点击"Deploy site"按钮开始部署过程。Netlify将克隆您的仓库，运行构建命令，并部署生成的文件。

步骤 4: 配置域名

网站部署完成后，Netlify会自动分配一个随机的子域名（如your-site-name.netlify.app）。您可以：

更改Netlify子域名

  1. 在站点设置中，找到"Domain settings"选项
  2. 点击"Change site name"按钮
  3. 输入新的子域名（无需包含.netlify.app）
  4. 点击"Save"保存更改

绑定自定义域名

  1. 在站点设置中，找到"Domain settings"选项
  2. 点击"Add custom domain"按钮
  3. 输入您的域名（如example.com）并点击"Verify"
  4. 按照提示在您的域名注册商处添加DNS记录
  5. 等待DNS记录生效（通常需要几分钟到几小时）

绑定域名后，Netlify会自动为您的网站配置SSL证书，确保访问安全。

步骤 5: 高级配置与功能

Netlify提供了多种高级功能，可以根据需要进行配置：

Serverless函数

在项目中创建netlify/functions目录，添加JavaScript或Go函数，实现后端功能无需服务器。

重定向和代理

创建netlify.toml文件配置URL重定向和代理规则，解决单页应用路由问题和API跨域问题。

表单处理

使用Netlify Forms收集表单数据，支持邮件通知和第三方集成，免费计划有提交数量限制。

构建环境变量

在站点设置中配置环境变量，安全地存储API密钥等敏感信息，在构建过程中使用。

完成这些配置后，您的网站将充分利用Netlify的功能，提供良好的用户体验。每次向您的Git仓库推送更改，Netlify都会自动重新构建和部署您的网站。
