What Does Algolia Code Look Like?

Developers use simple code to make Algolia work on websites. Here's a tiny example! 

// Create a search client
const searchClient = algoliasearch(
  'YOUR_APP_ID',
  'YOUR_SEARCH_API_KEY'
);

// Connect to your data
const index = searchClient.initIndex('games');

// When someone types in the search box
searchBox.addEventListener('input', (event) => {
  // Search for what they typed
  index.search(event.target.value).then(({ hits }) => {
    // Show the results!
    displayResults(hits);
  });
});

What this code does:

  • 1. Connects to Algolia's super fast search system
  • 2. Looks at a list of games (or whatever you're searching)
  • 3. Waits for someone to type in the search box
  • 4. Finds matching games and shows them instantly!
