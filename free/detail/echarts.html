Let's Play with ECharts!

Create colorful charts and awesome graphs with code! It's like magic for numbers!

See Cool Examples Play with Code
let myChart = echarts.init(document.getElementById('myDiv'));
 
// Let's make a colorful chart!
let option = {
  'title': {
    'text': 'My Favorite Snacks'
  },
  'series': [{
    'type': 'pie',
    'data': [
      {'name': 'Ice Cream', 'value': 35},
      {'name': 'Cookies', 'value': 25},
      {'name': 'Fruit', 'value': 40}
    ]
  }]
};
 
myChart.setOption(option);

What is ECharts?

Code Playground

Try changing the code and see what happens!

chart.js
let ctx = document.getElementById('myPlaygroundChart').getContext('2d');
 
let chart = new Chart(ctx, {
  type: 'bar',
  data: {
    labels: ['Mon', 'Tue', 'Wed', 'Thu', 'Fri'],
    datasets: [{
      label: 'Hours of Play',
      data: [2, 1, 3, 2, 4],
      backgroundColor: [
        'rgba(255, 107, 107, 0.7)',
        'rgba(78, 205, 196, 0.7)',
        'rgba(255, 209, 102, 0.7)',
        'rgba(156, 39, 176, 0.7)',
        'rgba(33, 150, 243, 0.7)'
      ]
    }]
  },
  options: {
    scales: {
      y: {
        beginAtZero: true
      }
    }
  }
});
