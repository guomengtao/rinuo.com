Let's Start

What is C++?

C++ is like a special language that tells computers what to do!

It's Everywhere!

C++ is used to make all sorts of cool things that kids love:

  • Video games like Minecraft and Roblox
  • Phone apps and tablets
  • Computer programs and software
  • Robots and cool machines
  • Smart TVs and game consoles

When you learn C++, you're learning how to create your own games and programs!

Fun to Learn!

How Does C++ Work?

It's like giving instructions to a very smart robot!

You Write Code

You type special words and symbols that the computer understands. It's like writing a secret message!

Computer Reads It

The computer translates your code into actions. It follows your instructions step by step!

Awesome Things Happen!

Your code makes the computer do cool stuff - like drawing pictures, playing games, or solving puzzles!

Fun Code Examples

Let's look at some simple C++ code and see what it does!

Saying Hello

This code makes the computer say "Hello, World!" Can you change it to say your name? 

#include <iostream>

int main() {
    // This line makes the computer talk!
    std::cout << "Hello, World!";
    
    // Every program needs this at the end
    return 0;
}

What It Does:

Hello, World!

Try It:

Counting Numbers

This code makes the computer count from 1 to 5. Can you make it count higher? 

#include <iostream>

int main() {
    // This counts from 1 to 5
    for (int i = 1; i <= 5; i++) {
        std::cout << i << " ";
    }
    
    return 0;
}

What It Does:

1 2 3 4 5

Try It:

Drawing Shapes

This code draws a square made of stars. Can you make it bigger or smaller? 

#include <iostream>

int main() {
    // This draws a square
    int size = 5;
    
    for (int i = 0; i < size; i++) {
        for (int j = 0; j < size; j++) {
            std::cout << "* ";
        }
        std::cout << std::endl;
    }
    
    return 0;
}
