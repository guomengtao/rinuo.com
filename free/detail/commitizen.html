Hello, Little Coder!

Let's learn about Commitizen!

1 What is Commitizen?

Commitizen is like a special helper for coders!

When we make changes to our code, we need to write a note about what we did.

Commitizen helps us write these notes in a clear and happy way!

Note!

2 Why Do We Use It?

Using Commitizen helps everyone understand what changed!

It's like putting labels on our toys so we know what each one does.

When notes are clear, working together is easier!

Fixed a bug!

Made the button work again

Added a cat picture

New cute image for babies

3 How to Use It?

Step 1: Make your changes

Write or change your code, just like building with blocks!

Step 2: Ask for help

Type git cz in your terminal. This calls Commitizen!

> git cz
? Select the type of change you're committing: (Use arrow keys)

Step 3: Answer questions

Commitizen will ask simple questions. Just pick the answers!

Feature Bug Fix Documentation Performance

Step 4: Done!

Commitizen will write a perfect note for you. Great job!

Let's See It in Action!

> git add .

> git cz

? Select the type of change you're committing: (Use arrow keys)

❯ feat: A new feature

fix: A bug fix

docs: Documentation only changes


? What is the scope of this change? (e.g. component or file name)

baby-app


? Write a short, imperative tense description of the change:

add colorful buttons


? Provide a longer description of the change:

I added big, colorful buttons that babies will love!


Perfect commit created!

This Creates a Note Like This:

feat(baby-app): add colorful buttons

I added big, colorful buttons that babies will love!

Everyone understands!

See? It's easy to tell what changed!

Let's Remember!

What?

Commitizen is a helper that helps us write notes about our code changes.

Why?

It makes our notes clear so everyone can understand what we did.

How?

Type "git cz" and answer simple questions. Commitizen does the rest!

You're ready to use Commitizen like a pro!
