What is Appwrite, Anyway?

Imagine you want to build a super cool game or app where people can sign up, save their scores, and share stuff with friends.

To do that, you'd need to figure out a lot of complicated things:

  • How to let people create accounts and log in
  • How to save scores or drawings so they don't disappear
  • How to make sure only certain people can see secret stuff
  • How to send messages between players

That's where Appwrite comes in! It does all that hard stuff for you, so you can focus on making your game or app awesome!

See It In Action

Check out these simple code examples that show how easy Appwrite is to use!

Creating a new user 
// Make a new account for someone
const user = await appwrite.account.create(
  "unique@email.com",
  "superSecretPassword123",
  "Cool Username"
);

// Show a message when it works!
if (user) {
  showMessage("Yay! New user created!");
}
Saving a high score 
// Save a high score in the game
const score = await appwrite.database.createDocument(
  "gameCollection",
  {
    player: "Cool Username",
    points: 1500,
    level: 7
  }
);

// Celebrate the high score!
if (score) {
  showMessage("Wow! New high score saved!");
}
