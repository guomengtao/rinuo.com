How Do I Use It?

It's as easy as 1-2-3! Let's see how simple it is to create your first site.

1

Install Docusaurus

Open your terminal and type this magic command:

npx create-docusaurus@latest my-website classic
2

Start Your Website

Move into your new website folder and start it up:

cd my-website
npm run start
3

Make It Yours

Edit the files to add your own content:

// Edit this file: src/pages/index.js
function Home() {
return <h1>My Awesome Website!</h1>
}
