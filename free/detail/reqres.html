What is Reqres?

Reqres is like a pretend friend for computer programs! It helps them practice talking to each other.

When programmers are learning, they can use Reqres to try out their code without worrying about making mistakes.

It's safe, fun, and perfect for learning how computers share information!

Example Code 
// Asking Reqres for information
fetch('https://reqres.in/api/users')
  .then(response => response.json())
  .then(data => {
    console.log("Look what I got!");
    console.log(data);
  });

How Does It Work?

You Send a Message

Your program sends a question or request to Reqres, like asking for a list of users.

Reqres Replies

Reqres sends back a response with the information you asked for, just like a friend writing back!

You Learn & Play

You see how the conversation works and can try different messages to learn more!
