What is Fig?

Fig is a powerful terminal enhancement tool designed to boost developer productivity. It adds modern features like context-aware autocomplete, command history search, and customizable themes to your existing terminal.

Unlike full terminal emulators, Fig works with your preferred terminal (like iTerm, Terminal, Warp, and others) by adding a layer of functionality on top. It understands the commands you're typing and provides relevant suggestions in real-time, helping you write commands faster and with fewer errors.

Fig also includes team collaboration features, allowing you to share commands and workflows with your team members. This makes it easier to standardize development processes and reduce knowledge silos.

How to Use Fig

Terminal

# Install Fig on macOS

brew install fig

# Or download directly from the website

open https://fig.io/download

# After installation, start Fig

fig start

See it in Action!

Here's how Fig works in your terminal:

1. Typing Commands

As you type commands, Fig will suggest completions based on what you've typed so far.

2. Using Tab Completion

Press Tab to accept suggestions or cycle through options. Fig understands the context of your command.

3. Creating Team Shortcuts

Save commonly used commands as team shortcuts that everyone can access.
