What is Vitest?

Vitest is like a super helper for people who write code! It checks if the code works correctly, just like a teacher checks your homework.

When programmers write code, they want to make sure it does exactly what it's supposed to do. Vitest helps them test their code quickly and easily!

What does it do?

Vitest runs special test code to make sure your main code works properly, even when you make changes!

Why is it cool?

It's super fast! You don't have to wait long to see if your code works. It's like a speed racer for tests!

How Does Vitest Work?

1

Write Your Code

First, you create your code - like making a fun game or a cool app!

2

Write Tests

Then you write special test code that tells Vitest what to check for.

3

Run Vitest

You tell Vitest to start checking, and it runs all your tests super fast!

4

See Results

Vitest shows you which tests passed (green) and which need fixing (red).

Let's See Vitest in Action!

Here's a simple example of how Vitest works. We'll test a function that adds numbers!

Our Code to Test:

// This is a simple function that adds two numbers
function addNumbers(a, b) {
  return a + b;
}

Our Vitest Test:

// This is our test file using Vitest
import { test, expect } from 'vitest';
import { addNumbers } from './our-code.js';

// Test if adding works correctly
test('adds two numbers together', () => {
  // Check if 2 + 3 equals 5
  expect(addNumbers(2, 3)).toBe(5);
  
  // Check if 10 + (-5) equals 5
  expect(addNumbers(10, -5)).toBe(5);
});

What Vitest Shows Us:

✓ adds two numbers together (5ms)
Test Files 1 passed (1)
Tests 1 passed (1)
Duration 12ms
