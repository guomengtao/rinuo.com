SQL in Action

Let's see how SQL works with a fun example!

Toy Store Database

Imagine we have a toy store with different toys. Let's use SQL to find what we want!

Toy Name Color Price
Teddy Bear Brown $10
Ball Red $5
Doll Pink $12

If we want to find all red toys, we can ask:

SELECT * FROM toys WHERE color = 'Red';

And SQL will give us:

Toy Name: Ball

Color: Red

Price: $5
