Let's Explore Yarn!

Discover the colorful world of yarn and all the amazing things you can create with it!

What is Yarn?

Yarn is a long, continuous length of interlocked fibers. It's like a colorful string that you can use to make all sorts of cool things!

Comes in Colors!

Yarn comes in every color you can imagine - bright, dark, sparkly, and even rainbow!

Different Sizes

Some yarn is thin like a thread, some is thick and fluffy - there's yarn for every project!

Fun to Use

You can wrap it, weave it, knit it, or crochet it into almost anything you can dream up!

How to Use Yarn

Using yarn is easy and fun! Try these simple ways to create with yarn.

1

Wrap It

Wrap yarn around objects to make colorful decorations. Try wrapping it around a toilet paper roll to make a binocular!

2

Glue It

Use school glue to attach yarn to paper. Make pictures by gluing yarn into different shapes!

3

Knit It (With Help)

With help from an adult, you can learn simple knitting. Start with a easy scarf or a small pouch!
