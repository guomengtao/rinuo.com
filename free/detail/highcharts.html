The Story Behind Highcharts

Project Origin

Highcharts was first released in 2009 by Torstein Hønsi. Frustrated by the lack of high-quality, interactive charting solutions for the web, Torstein set out to create a library that combined powerful functionality with ease of use.

What started as a side project quickly gained popularity among developers, eventually becoming one of the most widely used JavaScript charting libraries on the web.

Founder: Torstein Hønsi

Torstein Hønsi is a Norwegian software developer with a passion for data visualization. Before creating Highcharts, he worked as a web developer, where he recognized the need for better charting tools.

Under his leadership, Highcharts has grown from a one-person project to a successful company with a team of developers working on continuous improvements and new features.

Technology Stack

Highcharts is built primarily with JavaScript and uses SVG for rendering charts in modern browsers. It's framework-agnostic but offers official wrappers for:

  • React
  • Vue.js
  • Angular
  • TypeScript
  • jQuery

Resources & Getting Started

Everything you need to start using Highcharts in your projects.

Getting Started

Highcharts is available for free under the Creative Commons Attribution-NonCommercial license for non-commercial use. For commercial projects, you'll need to purchase a license.

Basic Implementation

<!-- Include Highcharts -->
<script src="https://code.highcharts.com/highcharts.js"></script>

<!-- Create a container -->
<div id="container" style="width:100%; height:400px;"></div>

<!-- Initialize chart -->
<script>
Highcharts.chart('container', {
    title: {
        text: 'My First Highchart'
    },
    series: [{
        data: [1, 3, 2, 4]
    }]
});
</script>
