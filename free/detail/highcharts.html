Let's Make Cool Charts!

With Highcharts, drawing pictures with numbers is easy!

What is Highcharts?

Highcharts is like a magic crayon box for numbers! It helps turn boring numbers into colorful pictures called "charts".

With Highcharts you can make:

  • Line charts that show how things change
  • Bar charts that compare different things
  • Pie charts that look like pizza slices
  • And many more fun picture types!

Try This Simple Example

See how easy it is to make your first chart. This one shows how many cookies were eaten each day!

// Let's make a cookie chart!
const chart = Highcharts.chart('cookieChart', {
  title: {
    text: 'Cookies Eaten This Week'
  },
  xAxis: {
    categories: ['Mon', 'Tue', 'Wed', 'Thu', 'Fri']
  },
  yAxis: {
    title: {
      text: 'Number of Cookies'
    }
  },
  series: [{
    name: 'Cookies',
    data: [3, 2, 4, 1, 5],
    color: '#ff9f1c'
  }]
});
