What is Mocha?

Mocha is like a magic tool that helps programmers check if their code works properly. It's fun and easy to use!

A Code Checker

Mocha helps make sure your code does what it's supposed to do. It's like a teacher checking your homework!

Fun to Use

Using Mocha feels like playing a game. You get to see if your code passes all the challenges!

Learn & Grow

With Mocha, you'll learn how to make better code. It's a great way to become a coding superstar!

Mocha in Action

See how Mocha works with these simple examples. It's like magic for your code!

Checking Numbers

Mocha can check if numbers are doing what they should!

// A simple function to add two numbers
function add(a, b) {
  return a + b;
}

// Mocha checks if it works!
mocha.check("Addition test", function() {
  mocha.assert(add(2, 3) === 5); // This should pass!
  mocha.assert(add(5, 5) === 10); // This too!
});

Checking Words

Mocha can also check words and sentences!

// A function to greet people
function greet(name) {
  return "Hello, " + name + "!";
}

// Mocha checks our greetings
mocha.check("Greeting test", function() {
  mocha.assert(greet("Alice") === "Hello, Alice!");
  mocha.assert(greet("Bob") === "Hello, Bob!");
});
