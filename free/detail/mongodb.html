Let's Explore MongoDB!

It's like a digital treasure chest!

MongoDB is where computers keep all their information. It's like a super organized toy box for data!

What is MongoDB?

A Digital Storage Box

MongoDB is like a special box where computers keep information - like your toys in your toy box, but for data!

Easy to Find Things

It's super easy to find what you're looking for in MongoDB, like having labels on all your toy boxes!

Grows When You Need It

MongoDB can get bigger and bigger as you add more information - like a toy box that magically grows!

How MongoDB Stores Information

MongoDB stores information in special containers called "documents". It's like putting different toys in clear plastic bags so you can see what's inside!

Document

A document is like a labeled bag that holds information. It can contain numbers, words, and even lists!

{
  "name": "Luna",
  "type": "Dog",
  "age": 3,
  "hobbies": ["fetch", "napping"]
}

Collection

A collection is like a shelf that holds many similar documents, just like a bookshelf holds books!

Document 1 (Luna)
Document 2 (Max)
Document 3 (Charlie)

Let's Try Adding Data!

Imagine we're collecting information about our friends. Let's add a new friend to our MongoDB collection:

MongoDB Shell
db.friends.insertOne({
    "name": "Alex",
    "age": 8,
    "favoriteColor": "blue",
    "hobbies": ["drawing", "soccer"]
})

Create your own friend entry:

Your Friend was Added!

{
  "name": "Sam",
  "age": 7,
  "favoriteColor": "green",
  "hobbies": ["reading"]
}

Great job! You just added data to MongoDB!
