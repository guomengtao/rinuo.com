What is Playwright?

Playwright is like a robot friend that can use websites just like humans do!

It can:

  • Click buttons and links on web pages
  • Type text into forms and search boxes
  • Take pictures of web pages
  • Check if websites are working correctly
  • Do all these things automatically, without anyone clicking!

Programmers use Playwright to make sure websites work properly, even when lots of people are using them at the same time.

Fun Fact!

Playwright was created by Microsoft, but anyone can use it for free. It works with all the popular web browsers like Chrome, Firefox, and Safari!

How Does It Work?

Terminal
$ npm install playwright
// Installing Playwright...
// Success! Playwright is ready to use
$ npx playwright codegen
// Recording your actions...
// Code generated successfully!
$ node my_first_test.js
// Running your test...
✓ Test passed!

It's that easy! With just a few commands, you can make Playwright do amazing things!

Cool Things You Can Do

Check out these examples of what Playwright can do. Try to understand them - you'll be writing your own soon!

Take a Screenshot

Playwright can take pictures of any website automatically!

const { chromium } = require('playwright');

(async () => {
  // Launch a browser
  const browser = await chromium.launch();
  const page = await browser.newPage();
  
  // Go to a website
  await page.goto('https://example.com');
  
  // Take a screenshot
  await page.screenshot({ path: 'example.png' });
  
  // Close the browser
  await browser.close();
})();

Fill Out Forms

Playwright can automatically fill out and submit forms for you!

const { firefox } = require('playwright');

(async () => {
  const browser = await firefox.launch();
  const page = await browser.newPage();
  
  // Go to a form page
  await page.goto('https://example.com/form');
  
  // Fill out the form
  await page.fill('#name', 'My Name');
  await page.fill('#email', 'me@example.com');
  
  // Click the submit button
  await page.click('button[type="submit"]');
  
  await browser.close();
})();
