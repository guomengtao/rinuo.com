Meet Jest, the Code Detective!

Jest checks if computer code works properly, like a detective solving mysteries!

jest --findMysteries=true 🔍

What Does Jest Do?

Imagine you built a cool treehouse. You'd want to check if:

- The ladder works
- The roof doesn't leak
- The door opens and closes

That's exactly what Jest does for computer programs! It's like a detective that:

Checks for Problems

Finds mistakes in code before anyone notices them, like a good detective!

Makes Sure It Works

Verifies that all parts of a program do exactly what they're supposed to do.

Warns About Issues

Tells programmers when something might break or work incorrectly.

Ensures Quality

Helps make programs reliable and awesome for everyone who uses them!

How Does Jest Solve Mysteries?

1. The Case is Defined

Programmers tell Jest what to check, like "Does the calculator add numbers correctly?"

1
2

2. Jest Investigates

Jest runs the code and checks if everything works like it should, looking for clues.

3. The Verdict is Given

Jest reports back with whether the code passed the test or needs fixing.

Let's Try Jest Detective Work!

Watch how Jest checks if a math function works correctly. Click "Run Tests" to see the detective work! 

// A simple math function
function addNumbers(a, b) {
  return a + b;
}

// Jest detective tests
test('adds 2 + 3 to equal 5', () => {
  expect(addNumbers(2, 3)).toBe(5);
});

test('adds 10 + 20 to equal 30', () => {
  expect(addNumbers(10, 20)).toBe(30);
});

test('adds 5 + 5 to equal 10', () => {
  expect(addNumbers(5, 5)).toBe(10);
});
