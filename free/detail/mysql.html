MySQL 版本历史

MySQL 8.0

2018年4月发布

引入窗口函数、CTE、JSON增强、原子DDL等重大特性，性能提升约2倍

MySQL 5.7

2015年10月发布

增强JSON支持、空间数据、优化器改进、安全性提升，InnoDB性能增强

MySQL 5.6

2013年2月发布

引入全文搜索、时间点恢复、InnoDB性能大幅提升，优化器改进
