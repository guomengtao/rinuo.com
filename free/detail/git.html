What is Git?

Git is like a magic save button for your computer projects!

When you make something cool - a drawing, a game, or a story - Git helps you:

  • Save your work so you never lose it
  • Keep track of all your changes
  • Share your projects with friends
  • Try new ideas without breaking your work

How Git Works - Step by Step

1

Create

Make something awesome on your computer

2

Save

Tell Git to save your work with a message

3

Share

Send your project to friends to see and play with

Let's Try with a Story!

Day 1: You start a story about a dragon. You save it with Git.

Day 2: You add a princess to your story. Git saves this new version.

Day 3: You change the dragon to be friendly. Git saves this too!

Later: You can see all your different versions, or even go back to any day's story!

That's what Git does! It keeps all your versions safe and sound.
