Let's Make a Colorful Square!

This code will create a square that changes colors. Click "Run" to see it!

>
// Let's create a square
let square = makeSquare();
square.setSize(100);
square.setColor("red");
square.onClick(function() {
square.changeColor();
});
// Click the square to change colors!
