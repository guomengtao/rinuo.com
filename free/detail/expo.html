What is Expo?

App Builder Tool

Expo is a super cool tool that helps you create your own mobile apps for phones and tablets!

Coding Made Easy

You can make apps with simple code, and see your creations right away on your phone!

Fast & Fun

Expo helps you build apps quickly, so you can turn your ideas into real apps in no time!

Works Everywhere

Apps you make with Expo work on iPhones, Android phones, and even on the web!

How to Use Expo

Set Up Expo

First, you need to install Expo on your computer and phone. It's quick and easy!

1
npm install -g expo-cli
expo init my-cool-app
cd my-cool-app
expo start
2

Create Your First App

Use simple commands to create a new app project and start working on it.

Edit Your App

Change the code to make your app do cool things. You can add buttons, pictures, and more!

3
// Change this in App.js
<Text>Hello, World!</Text>

Scan the QR code with your Expo app to see your creation!

4

See Your App

Watch your app come to life on your phone instantly as you make changes!

Try a Simple Expo Demo

See how easy it is to change code and make something cool!

Change the Text:

import React from 'react';
import { View, Text, StyleSheet } from 'react-native';

export default function App() {
  return (
    <View style={styles.container}>
      <Text style={styles.text}>
        Hello, Expo!
      </Text>
    </View>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
    backgroundColor: '#fff',
  },
  text: {
    fontSize: 24,
    fontWeight: 'bold',
    color: '#4285F4',
  },
});
