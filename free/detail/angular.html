What is Angular, anyway?

Think of Angular as a set of building blocks for creating amazing websites and apps!

Building Blocks

Angular gives you ready-made pieces to build with, like Lego blocks for websites!

Super Fast

Apps made with Angular run really fast, even when they do lots of cool stuff!

Everywhere

You can use Angular to make websites, phone apps, and even games!

Hello, Friend! What's your name?

<!-- This is the Angular template -->
<input [(ngModel)]="yourName" placeholder="Type your name">
<h4>Hello, {{ yourName || 'Friend' }}!</h4>

// This is the component code
export class GreetingComponent {
  yourName: string = '';
}
