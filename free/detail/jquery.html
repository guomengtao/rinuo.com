jQuery 使用方法

快速入门 jQuery，从安装到基本操作

步骤 1: 引入 jQuery

有多种方式可以引入 jQuery 到你的项目中：

通过 CDN 引入（推荐）

<!-- 使用 jQuery 官方 CDN -->
<script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>

<!-- 或者使用 Google CDN -->
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.7.1/jquery.min.js"></script>

CDN 可以提高加载速度，减少服务器负担

下载到本地引入

<!-- 从官网下载后引入本地文件 -->
<script src="jquery-3.7.1.min.js"></script>

下载地址: https://code.jquery.com/

使用 npm 安装

npm install jquery

// 在项目中引入
import $ from 'jquery';

适合现代前端工程化项目

步骤 2: jQuery 基本语法

jQuery 的基本语法是通过选择器选取 HTML 元素，并对选取的元素执行某些操作：

基础语法结构

$(selector).action()
  • $ 符号定义 jQuery
  • selector 用于"查询"和"查找" HTML 元素
  • action() 用于执行对元素的操作

文档就绪事件

// 确保DOM加载完成后再执行代码
$(document).ready(function(){
  // jQuery 代码...
  $("p").hide(); // 隐藏所有段落
});

// 简写形式
$(function(){
  // jQuery 代码...
});

这是最安全的方式，可以避免在DOM元素加载完成前执行代码

常用选择器示例

// 元素选择器 - 选取所有 <p> 元素
$("p")

// ID选择器 - 选取 id="demo" 的元素
$("#demo")

// 类选择器 - 选取 class="test" 的所有元素
$(".test")

// 属性选择器 - 选取所有带有 href 属性的元素
$("[href]")

步骤 3: DOM 操作

jQuery 提供了简洁的方法来操作 HTML 元素和属性：

内容操作

// 获取/设置元素的文本内容
$("#test").text(); // 获取
$("#test").text("新文本"); // 设置

// 获取/设置元素的HTML内容
$("#test").html(); // 获取
$("#test").html("<b>新HTML</b>"); // 设置

// 获取/设置表单字段的值
$("input").val(); // 获取
$("input").val("新值"); // 设置

属性操作

// 获取/设置属性值
$("img").attr("src"); // 获取
$("img").attr("src", "new-image.jpg"); // 设置

// 设置多个属性
$("a").attr({
  "href": "https://jquery.com",
  "title": "jQuery官方网站"
});

// 移除属性
$("img").removeAttr("src");

CSS 操作

// 添加CSS类
$("p").addClass("highlight");

// 移除CSS类
$("p").removeClass("highlight");

// 切换CSS类
$("p").toggleClass("highlight");

// 获取/设置CSS属性
$("p").css("color"); // 获取
$("p").css("color", "red"); // 设置
$("p").css({"color": "red", "font-size": "16px"}); // 设置多个

步骤 4: 事件处理

jQuery 提供了简洁的方式来处理各种事件，如点击、鼠标移动、表单提交等：

常用事件示例

$(document).ready(function(){
  // 点击事件
$("button").click(function(){
    alert("按钮被点击了");
  });

  // 双击事件
$("p").dblclick(function(){
    $(this).hide();
  });

  // 鼠标进入事件
$("div").mouseenter(function(){
    $(this).css("background-color", "lightgray");
  });

  // 鼠标离开事件
$("div").mouseleave(function(){
    $(this).css("background-color", "white");
  });
});

事件绑定与解绑

// 绑定事件
$("button").on("click", function(){
  $(this).text("已点击");
});

// 绑定多个事件
$("input").on({
  focus: function(){ $(this).css("background-color", "lightblue"); },
  blur: function(){ $(this).css("background-color", "white"); },
  change: function(){ $(this).css("background-color", "lightgreen"); }
});

// 解绑事件
$("button").off("click");

事件委托

// 为动态添加的元素绑定事件
$("#container").on("click", "p", function(){
  $(this).slideToggle();
});

// 动态添加元素
$("button").click(function(){
  $("#container").append("<p>新段落</p>");
});

事件委托允许我们为尚不存在的元素绑定事件处理程序

步骤 5: AJAX 操作

jQuery 简化了 AJAX 操作，提供了简洁的方法来与服务器交换数据：

$.ajax() 方法

$.ajax({
  url: "data.json", // 请求的URL
  type: "GET", // 请求方法
  dataType: "json", // 预期的响应数据类型
  success: function(data) {
    // 请求成功时执行
    console.log("数据加载成功:", data);
    $("#result").html(data.content);
  },
  error: function(xhr, status, error) {
    // 请求失败时执行
    console.error("请求失败:", error);
  },
  complete: function() {
    // 请求完成时执行（无论成功或失败）
    console.log("请求完成");
  }
});

简化的 AJAX 方法

// 加载HTML内容
$("#div1").load("content.html");

// GET请求
$.get("demo.php", function(data, status){
  alert("数据: " + data + "\n状态: " + status);
});

// POST请求
$.post("demo.php",
  { name: "John", age: 30 },
  function(data, status){
    alert("数据: " + data + "\n状态: " + status);
  }
);
