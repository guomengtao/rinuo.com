LESS Makes Styling Super Easy!

When you want to make websites look cool, LESS helps you:

  • Create colors once and use them everywhere
  • Make changes to your design super fast
  • Write less code but do more awesome things
  • Create amazing styles without repeating yourself

Look at this magic!

With LESS, you can create a color once and use it everywhere. When you change it once, it changes everywhere!

@myColor: #E63946;

.button { color: @myColor; }

.title { border-color: @myColor; }

How Do You Use LESS?

It's easy! LESS adds superpowers to regular CSS:

Regular CSS

.box {

color: #E63946;

background: #F1FAEE;

}

.button {

color: #E63946;

border: 2px solid #E63946;

}

Super LESS

@brandColor: #E63946;

@bgColor: #F1FAEE;

.box {

color: @brandColor;

background: @bgColor;

}

.button {

color: @brandColor;

border: 2px solid @brandColor;

}
