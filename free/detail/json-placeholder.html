About JSONPlaceholder

A simple yet powerful tool for developers worldwide

What is JSONPlaceholder?

JSONPlaceholder is a free online REST API that provides fake data for developers. It's perfect for testing new libraries, prototyping applications, creating tutorials, or learning new technologies.

With endpoints for posts, comments, users, albums, photos, and todos, it covers common data structures you'll encounter in real-world applications.

Project Origin

JSONPlaceholder was created in 2013 out of frustration with the lack of simple, free APIs for testing. The goal was to provide developers with an instantly available resource that doesn't require registration or authentication.

Founder

Typicode

A French developer passionate about simplifying development workflows. Typicode has created several popular open-source tools including Husky and lowdb.

Technical Stack

The technology powering JSONPlaceholder

Node.js & Express

The core API is built with Node.js and Express, providing a lightweight and efficient server that can handle thousands of concurrent requests.

JSON Server

Under the hood, JSONPlaceholder uses JSON Server to provide a full REST API from a simple JSON file, making it easy to maintain and extend.

Cloud Hosting

Hosted on multiple cloud providers with CDN distribution to ensure fast response times worldwide, even during peak usage.

The Good and The Bad

What makes JSONPlaceholder shine, and where it falls short

Advantages

  • Completely free to use with no rate limits
  • No authentication or registration required
  • Supports all standard REST operations (GET, POST, PUT, DELETE)
  • Realistic data structures that mimic real-world applications
  • Blazing fast response times with global CDN distribution
  • Great documentation and examples for quick onboarding

Limitations

  • Data is not persistent - changes reset periodically
  • Limited data models compared to specialized APIs
  • Not suitable for production environments
  • No advanced query features like complex filtering
  • Occasional downtime during maintenance periods
  • Limited to JSON format, no XML or other formats

Get Started in Seconds

It's never been easier to start using fake data

Simple Example

Just make a GET request to any endpoint. No setup or authentication required: 

// Using fetch API
fetch('https://jsonplaceholder.typicode.com/posts/1')
  .then(response => response.json())
  .then(json => console.log(json))

// Using axios
axios.get('https://jsonplaceholder.typicode.com/posts/1')
  .then(response => console.log(response.data))

// Using curl
curl https://jsonplaceholder.typicode.com/posts/1

Response:

{
  "userId": 1,
  "id": 1,
  "title": "sunt aut facere repellat provident occaecati excepturi optio reprehenderit",
  "body": "quia et suscipit\nsuscipit recusandae consequuntur expedita et cum\nreprehenderit molestiae ut ut quas totam\nnostrum rerum est autem sunt rem eveniet architecto"
}

Available Endpoints

  • /posts - 100 posts
  • /comments - 500 comments
  • /users - 10 users
  • /albums - 100 albums
  • /photos - 5000 photos
  • /todos - 200 todos

Supported Methods

  • GET Retrieve resources
  • POST Create new resources
  • PUT Update existing resources
  • DELETE Remove resources

Self-Hosting

For offline use or custom data, you can host JSONPlaceholder yourself:

npm install -g json-server
json-server --watch db.json
