Let's Play with JSON!

JSONPlaceholder is like a magical toy box full of pretend data for kids who love coding! 

// Let's get some pretend data!
fetch('https://jsonplaceholder.typicode.com/posts/1')
  .then(response => response.json())

Explore JSON Magic

What's JSONPlaceholder?

It's a special website that gives us pretend data to play with when we're learning to code!

Pretend Data

It has fake posts, users, and comments - like a pretend social media!

Coding Practice

Perfect for practicing how to get and use data in your programs!

Always Works

You can use it anytime, anywhere - no setup required!

How Does It Work?

It's easy! Let's look at some simple examples.

Getting a Fake Post

This code gets a pretend blog post:

fetch('https://jsonplaceholder.typicode.com/posts/1') .then(response => response.json()) .then(data => showPost(data))

What You Get:

{
  "userId": 1,
  "id": 1,
  "title": "sunt aut facere",
  "body": "quia et suscipit..."
}

Getting Fake Users

This code gets a list of pretend people:

fetch('https://jsonplaceholder.typicode.com/users') .then(response => response.json()) .then(users => showUsers(users))
