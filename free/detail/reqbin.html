Let's Get Started

What is ReqBin?

Imagine you want to ask a friend for their favorite game. You send them a message, and they send one back.

Computers do the same thing! They send messages called "requests" to each other. ReqBin is like a special notebook that helps you see these messages.

With ReqBin, you can:

  • See what messages computers send
  • Learn how websites get information
  • Practice sending your own computer messages
// This is like a computer asking for information
GET /games/favorite HTTP/1.1
Host: friend.com
// This is the answer back
200 OK
Content-Type: text/plain
My favorite game is Minecraft!

How Does It Work?

1

You Create a Message

2

Click send! Your message travels through the internet to the other computer.

3

See the Reply

Watch as the other computer sends back a reply, just like getting mail!
