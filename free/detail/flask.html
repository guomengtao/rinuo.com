Let's Try Using Flask!

Step 1: Write Your First Flask App

Copy this code into a file called my_app.py

from flask import Flask

app = Flask(__name__)

@app.route("/")
def hello():
    return "Hi there! I made this with Flask! üöÄ"

if __name__ == "__main__":
    app.run(debug=True)

Step 2: Run Your App

Type this in your terminal:

python my_app.py

Step 3: See Your Creation!

Open your web browser and go to: http://localhost:5000
