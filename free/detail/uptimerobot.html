Your friendly robot that watches over websites while you sleep, play, or eat ice cream!

What is UptimeRobot Anyway?

Imagine you have a lemonade stand. You want to know if it's open or closed, right? UptimeRobot does the same thing for websites!

It checks websites 24/7, like a super dedicated guard, to make sure they're working properly. If something goes wrong, it sends a message to the website owner.

How Does It Work?

It's as easy as 1-2-3, like making a peanut butter and jelly sandwich!

1

Tell It What to Watch

Give UptimeRobot the address of the website you want to monitor, like giving a friend your phone number.

2

It Starts Watching

UptimeRobot checks the website all day and night, just like a lighthouse watches over the ocean.

3

It Sends Alerts

If something goes wrong, it tells the website owner immediately, like yelling "Hey, your ice cream is melting!"

Simple Code Example

This is how programmers set up UptimeRobot (don't worry, you don't need to understand all of it!)

setting-up-uptimerobot.js 
// Create a new monitor
const myMonitor = new UptimeRobot.Monitor({
  friendly_name: "My Awesome Website",
  url: "https://my-awesome-website.com",
  type: "HTTP",
  check_interval: 5 // Check every 5 minutes
});

// Set up alerts
myMonitor.on("down", () => {
  console.log("Oh no! The website is down!");
  sendAlertToOwner();
});

myMonitor.on("up", () => {
  console.log("Yay! The website is working again!");
});

// Start monitoring
myMonitor.start();
