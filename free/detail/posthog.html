What is PostHog?

PostHog is like a friendly detective that watches (in a good way!) how kids use websites and games.

It learns what you like and what's hard to use, then helps grown-ups make the website better!

  • Notices what you click on and like
  • Helps fix things that are confusing
  • Makes websites more fun to use

How Does PostHog Work?

1. You Play & Click

When you use a website or game, PostHog notices what you do - like which buttons you click.

2. It Collects Clues

PostHog gathers all these little clues about what you like and how you use the site.

3. Makes It Better

Grown-ups use these clues to fix problems and make the website more fun for you!

How PostHog Talks to Computers

This is simple code that shows how PostHog works (in kid-friendly language!): 

// PostHog's simple code
function learnFromUsers(userActions) {
  // Watch what kids do on the website
  let favoriteButtons = findMostClicked(userActions);
  let trickyParts = findWhereUsersGetStuck(userActions);
  
  // Make a report for grown-ups
  return {
    kidsLove: favoriteButtons,
    needsFixing: trickyParts,
    // Ideas to make the site better
    coolIdeas: suggestImprovements(favoriteButtons, trickyParts)
  };
}

// Let's try it with today's players
let todayActions = [
  { clicked: "Puzzle Game", times: 24 },
  { clicked: "Racing Game", times: 18 },
  { clicked: "Drawing Game", times: 32 }
];

// What did PostHog learn?
let report = learnFromUsers(todayActions);
console.log("Kids love the " + report.kidsLove[0] + " the most!");
