What is Cloudinary?

Cloudinary is a comprehensive media management platform that provides powerful APIs and tools for uploading, storing, managing, transforming, optimizing, and delivering images, videos, and other media assets.

With Cloudinary, you can easily manipulate media on the fly using URL parameters, automate image and video optimization, and deliver content through a fast CDN. It integrates seamlessly with popular frameworks and programming languages, making it a go-to solution for developers building media-rich applications.

Cloudinary helps improve your application's performance by delivering optimized media assets, reducing bandwidth usage, and providing a better user experience with faster load times.

How to Use Cloudinary

Terminal

# Install Cloudinary SDK

npm install cloudinary --save

# For React applications

npm install @cloudinary/react @cloudinary/url-gen --save

# For Next.js applications

npm install next-cloudinary --save

See it in Action!

Here's a basic Node.js example using Cloudinary:

const cloudinary = require('cloudinary').v2;

// Configure Cloudinary
cloudinary.config({
  cloud_name: 'your-cloud-name',
  api_key: 'your-api-key',
  api_secret: 'your-api-secret'
});

// Upload an image
async function uploadImage() {
  try {
    const result = await cloudinary.uploader.upload('sample.jpg', {
      public_id: 'sample-image'
    });
    console.log(result.secure_url);
    
    // Generate a transformed URL
    const transformedUrl = cloudinary.url('sample-image', {
      width: 300,
      height: 300,
      crop: 'fill',
      quality: 'auto'
    });
    console.log(transformedUrl);
  } catch (error) {
    console.error(error);
  }
}

uploadImage();
