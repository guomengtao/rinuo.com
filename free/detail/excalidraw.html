About Excalidraw

What is Excalidraw?

Excalidraw is a free and open-source diagramming tool that lets you create beautiful hand-drawn style illustrations with ease. It's perfect for sketching diagrams, wireframes, flowcharts, and more.

The tool focuses on simplicity and intuitiveness, providing a minimal interface that doesn't distract from your creative process while still offering powerful features.

Project Origin

Excalidraw was created in 2020 by Christopher Chedeau (also known as Vjeux), a software engineer at Facebook. Frustrated by the lack of simple tools for creating hand-drawn style diagrams quickly, he built the initial version of Excalidraw.

The project quickly gained popularity within the developer community for its simplicity and unique aesthetic, leading to a thriving open-source community that continues to expand its capabilities.

Technology Stack

Frontend Technologies

  • JavaScript - Core programming language
  • TypeScript - For type safety and better developer experience
  • React - UI library for building the user interface
  • Canvas API - For rendering the drawing surface

Additional Tools

  • WebRTC - For real-time collaboration features
  • IndexedDB - For local storage of drawings
  • Vercel - For hosting the web application
  • Yjs - For CRDT-based collaborative editing

Excalidraw Feature Comparison

Pros & Cons

Advantages

  • Beautiful hand-drawn aesthetic that feels more personal than sterile diagrams
  • Completely free and open-source with no hidden costs
  • Simple, intuitive interface that doesn't get in your way
  • Excellent real-time collaboration features
  • Works entirely in the browser with no installation required

Limitations

  • Not suitable for highly detailed technical diagrams requiring precision
  • Limited export options compared to professional tools
  • Some advanced features require familiarity with keyboard shortcuts
  • Offline functionality is limited without setting up local environment
  • Not ideal for complex 3D diagrams or highly stylized illustrations

How to Get & Use Excalidraw

  1. 1

    Access via Web Browser

    Simply visit excalidraw.com - no installation required. Your drawings are saved automatically in your browser.

  2. 2

    Install Desktop App

    Download the official desktop app for Windows, macOS, or Linux from the GitHub releases page.

  3. 3

    Self-Hosted Instance

    Deploy your own instance using the source code for enhanced privacy and control.

  4. 4

    Integrate with Other Tools

    Use Excalidraw extensions for VS Code, Obsidian, and other productivity tools for seamless workflow integration.
