What is Jupyter, anyway?

Jupyter is like a magic notebook for code! It's a special place where you can write computer instructions (called "code"), add pictures, write notes, and see what happens when you run your code — all in one place!

Imagine a notebook where you can write a story, draw a picture, and then make something happen with code — that's Jupyter!

How Does It Work?

1

Create Cells

Jupyter notebooks are made of "cells" — little boxes where you can type either code or notes.

2

Write Code

Type your code into a code cell. You can use different programming languages like Python!

3

Run & See!

Click run to execute your code. You'll see the results right away below your code!
