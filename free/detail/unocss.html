What is UnoCSS?

UnoCSS is a tool that helps you style websites quickly and easily. Instead of writing long code, you can use simple class names to make your website look awesome!

Think of it like using building blocks - each class is a block that changes how something looks. You can mix and match them to create anything you want!

<!-- This is a button with UnoCSS classes -->
<button class="bg-blue-500 hover:bg-blue-600 text-white 
               py-2 px-4 rounded-lg shadow">
  Click Me!
</button>
Click Me!

Basic UnoCSS Magic

Colors

Change colors easily with classes like bg-red-500 (background) or text-blue-400 (text).

Sizes

Control sizes with classes like w-20 (width) or h-16 (height).

Spacing

Add space with m-4 (margin) or p-3 (padding) to make things look neat.

Small padding
Bigger padding

Effects

Add cool effects with rounded-full (circles) or shadow-lg (shadows).
