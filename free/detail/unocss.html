What is UnoCSS?

UnoCSS is a modern atomic CSS engine that generates styles on demand. Unlike traditional CSS frameworks that ship with pre-defined utilities, UnoCSS works backwards - it scans your code for class names and generates the corresponding CSS dynamically.

This approach results in a much smaller CSS footprint and faster load times, while still giving you all the power and flexibility of atomic CSS.

Example 
<!-- Your HTML -->
<div class="text-center text-blue-500 hover:text-blue-700 transition-colors">
  Hello UnoCSS!
</div>

<!-- Generated CSS (automatically) -->
.text-center { text-align: center; }
.text-blue-500 { color: #3B82F6; }
.hover\:text-blue-700:hover { color: #1D4ED8; }
.transition-colors { transition-property: color; }

Origin & Creator

Project Origins

UnoCSS emerged in 2021 as an experiment to push the boundaries of atomic CSS. The project started as a side experiment by Anthony Fu, exploring whether a CSS engine could be built that's both extremely fast and highly customizable.

Drawing inspiration from Tailwind CSS, Windi CSS, and other atomic frameworks, UnoCSS aimed to address performance issues while providing more flexibility through its unique "preset" system.

What began as an experiment quickly gained traction in the web development community due to its impressive performance metrics and innovative approach to CSS generation.

Anthony Fu

Creator of UnoCSS

Anthony Fu is a prominent figure in the Vue.js community and a core team member at Vue. He's known for his work on various open-source projects including VueUse, VitePress, and the official Vue.js documentation.

Pros & Cons

Advantages

  • Blazing fast performance with on-demand generation
  • Extremely small production CSS size
  • Highly customizable through presets and rules
  • Zero runtime overhead
  • Works with any framework or vanilla HTML
  • Support for CSS-in-JS-like patterns

Disadvantages

  • Smaller ecosystem compared to established frameworks
  • Less mature documentation for complex use cases
  • Requires more configuration for beginners
  • IDE extensions are not as refined
  • Steeper learning curve for custom configurations

Get Started with UnoCSS

Ready to try UnoCSS? Follow these simple steps to integrate it into your project.

Installation

npm install unocss --save-dev

Basic Configuration

// vite.config.js
import UnoCSS from 'unocss/vite'

export default {
  plugins: [
    UnoCSS({ /* options */ })
  ]
}

Import in Your Project

// main.js
import 'uno.css'
