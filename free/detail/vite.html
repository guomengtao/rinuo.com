Let's Get Started

What is Vite?

Vite (say "veet") is a tool that helps people build websites super quickly!

It's like having a magic helper that makes creating websites faster and more fun.

Before Vite, building websites could be slow. Now, with Vite, you can see your changes instantly!

Super Fast

Vite starts up in a blink and shows your changes right away. No more waiting!

Easy to Use

Simple commands get you started quickly, even if you're new to building websites.

How to Use Vite

Terminal

# Create a new Vite project

npm create vite@latest my-awesome-project

# Go into your new project folder

cd my-awesome-project

# Install magic tools

npm install

# Start your project!

npm run dev

See it in Action!

When you run npm run dev, something amazing happens:

Local Server Running ✓ Active

Your website is now running here:

http://localhost:5173

Now you can make changes to your files, and Vite will automatically show them!
