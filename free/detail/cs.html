What is C#?

C# (say "C sharp") is a special language that programmers use to talk to computers!

With C#, you can make:

  • Fun video games like Minecraft!
  • Cool phone apps
  • Awesome computer programs
  • And so much more!

It's like giving instructions to a very smart robot that will do exactly what you tell it!

"C# is easy to learn and super powerful. Many kids just like you use it to create amazing things!"

How Programmers Use C#

Make Games

Create your own video games with colorful graphics and fun gameplay!

Build Apps

Design apps for phones and tablets that can do amazing things!

Control Robots

Write code that makes robots move, dance, and follow your commands!

Make the computer say hi!

Math Fun

Let the computer do math!

using System;

class Program
{
    static void Main()
    {
        int a = 5;
        int b = 7;
        int sum = a + b;
        
        Console.WriteLine(a + " + " + b + " = " + sum);
    }
}
