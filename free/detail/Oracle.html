What is Oracle?

Oracle is like a super smart digital helper that stores information and helps programs work!

It's Like a Magic Toy Box!

Imagine you have a toy box that can remember where all your toys are, even when you have hundreds of them! That's what Oracle does with information.

Programmers use Oracle to store all kinds of things - like scores in games, names of friends, or even pictures!

It helps computers work faster and remember things, just like how you remember your favorite games and songs!

Try Some Code!

Let's play with simple code that even computers can understand!

oracle_fun.js

// This is a comment - computers ignore these!
// Let's make Oracle say hello

function sayHello(name) {
    // This stores our message
    let message = "Hello, " + name + "! Welcome to Oracle!";
    
    // This shows our message
    document.getElementById("oracleMessage").textContent = message;
    
    // Let's add some color fun!
    changeColor();
}

function changeColor() {
    // Random color magic!
    const colors = ["#FF6B6B", "#4ECDC4", "#FFD166", "#7B68EE"];
    const randomColor = colors[Math.floor(Math.random() * colors.length)];
    
    document.getElementById("oracleMessage").style.color = randomColor;
}
