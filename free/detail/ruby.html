What is Ruby, Anyway?

Ruby is a programming language - a special way to talk to computers and make them do amazing things!

Easy to Read

Ruby looks almost like English! It's designed to be friendly and easy to understand, even if you're new to coding.

Fun to Use

With Ruby, you can make games, build websites, or create cool programs in just a few lines of code!

Lots of Friends

Many people love Ruby! There are tons of resources and helpful communities to help you learn.

Math Magic

Ruby is great at solving math problems!

puts "5 + 3 = " + (5 + 3).to_s
puts "10 - 4 = " + (10 - 4).to_s
puts "7 * 6 = " + (7 * 6).to_s
puts "20 / 2 = " + (20 / 2).to_s
Try changing the numbers!

→ 5 + 3 = 8

→ 10 - 4 = 6

→ 7 * 6 = 42

→ 20 / 2 = 10

Loop the Loop

Make Ruby repeat things automatically!

3.times do
  puts "Ruby is awesome!"
end

puts "---"

count = 1
while count <= 5
  puts "Count: " + count.to_s
  count = count + 1
end
Try changing the numbers 3 and 5!

→ Ruby is awesome!

→ Ruby is awesome!

→ Ruby is awesome!

→ ---

→ Count: 1

→ Count: 2

→ Count: 3

→ Count: 4

→ Count: 5
