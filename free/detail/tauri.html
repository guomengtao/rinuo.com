What is Tauri?

Tauri is an open-source framework for building cross-platform desktop applications using web technologies like JavaScript, HTML, and CSS.

Unlike traditional Electron apps that bundle a full browser, Tauri uses the native webview of the operating system, making apps much smaller and more secure.

Tauri combines a Rust backend with your favorite frontend framework to create truly native desktop experiences.

Tiny Apps

Tauri apps are typically 2-3MB in size, compared to 50-100MB for Electron apps.

Super Secure

Built with Rust and minimal dependencies, Tauri apps have excellent security properties.

How to Use Tauri

Terminal

# First, install Rust (required for Tauri)

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Create a new Tauri project with React

npm create tauri-app@latest my-tauri-app -- --template react-ts

# Run the development server

cd my-tauri-app && npm run tauri dev

# Build the production app

npm run tauri build
