What is Docker, Anyway?

It's not about shipping or whales (well, maybe a little about whales!)

Cool, right?

Think of Docker as...

A Lunch Box

It holds everything you need for your meal - sandwich, fruit, drink. Docker containers hold everything code needs to run!

Toy Storage Bins

You keep dolls in one bin, cars in another. Docker keeps different programs separate but easy to find.

Magic Moving Boxes

You can take your Docker box from home to school to a friend's house, and it works the same everywhere!

How Docker Works

It's like a fun game with simple steps!

1

Build a Container

Programmers create a special recipe (called a Dockerfile) that tells Docker what to put in the container.

FROM happy/program
ADD fun_code /myapp
RUN make_everything_work
2

Put It All Inside

Docker takes the recipe and puts everything needed into a neat little container - code, toys, and all!

3

Play Anywhere!

You can take your container anywhere - to school, a friend's house, or even across the world, and it works the same!

docker run my_awesome_container
// Wow! It works perfectly!
