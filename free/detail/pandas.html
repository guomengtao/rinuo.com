What is Pandas?

A Fun Data Tool!

Pandas is like a super helper for working with data. It can:

  • Organize information in neat tables
  • Find interesting patterns in numbers
  • Sort and filter information easily
  • Create colorful charts and graphs

A Simple Example:

# Import pandas

import pandas as pd


# Create a data table

data = {'Fruits': ['Apple', 'Banana', 'Orange'],

   'Count': [5, 3, 7]}


# Make a pandas DataFrame

df = pd.DataFrame(data)


# Show the table

print(df)
