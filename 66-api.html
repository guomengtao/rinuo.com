API接口手册

本手册详细介绍了所有可用的API接口分类及调用方法，帮助开发者快速集成我们提供的各类数据服务。所有接口均以JSON格式返回数据，支持跨域请求。

通用调用说明

所有接口均采用RESTful设计风格，返回数据格式为JSON。基础调用地址为：

https://api.example.com/v1/{category}

请求方式

  • GET：获取资源
  • POST：创建资源
  • PUT：更新资源
  • DELETE：删除资源

请求头

  • Content-Type: application/json
  • Authorization: Bearer {your_api_key}（如需认证）
  • X-Request-Id: 可选，用于追踪请求

通用响应格式

{ "success": true, "code": 200, "message": "操作成功", "data": { // 具体数据 }, "timestamp": 1629260863000, "requestId": "req-123456789" }

错误码说明

错误码 说明
200 成功
400 请求参数错误
401 未授权或token过期
403 权限不足
404 资源不存在
500 服务器内部错误

天气接口 weather.json

返回顶部

接口说明

提供全球城市天气查询服务，包括实时天气、未来预报、空气质量等信息。数据来源为多个权威气象数据源聚合。

基础地址

https://api.example.com/v1/weather

可用接口列表

实时天气查询
GET
接口地址：
/current
请求参数：
参数名 类型 是否必须 说明
city string 是 城市名称，如"北京"
lang string 否 语言，默认"zh-CN"
响应示例：
{ "success": true, "code": 200, "message": "操作成功", "data": { "city": "北京", "country": "中国", "temperature": 25, "feels_like": 27, "weather": "晴", "weather_code": "sunny", "humidity": 50, "wind_speed": 3.5, "wind_direction": "东北", "pressure": 1013, "visibility": 10, "uv_index": 6, "update_time": "2023-08-15T14:30:00+08:00" }, "timestamp": 1629260863000 }
未来7天天气预报
GET
接口地址：
/forecast/7days
请求参数：
参数名 类型 是否必须 说明
city string 是 城市名称，如"上海"

使用限制

  • 免费用户：每小时最多60次请求
  • 付费用户：根据套餐不同，每小时1000-10000次请求
  • 支持跨域请求
  • 数据缓存时间：10分钟

可用接口列表

足球赛事实时比分
GET
接口地址：
/football/live
请求参数：
参数名 类型 是否必须 说明
league string 否 联赛名称，如"premier-league"
team string 否 球队名称，如"manchester-united"
